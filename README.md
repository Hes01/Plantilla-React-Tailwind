## Installation

For Development...

```sh
git clone [https://github.com/akhshyganesh/React-Tailwind.git](https://github.com/Hes01/Plantilla-React-Tailwind.git
cd Plantilla-React-Tailwind
npm install
npm run dev
```

For production...

```sh
npm run build
```
